# Udemy-clone
this is the Udemy clone website I built this for my week 1 assignment in learning HTML and CSS 
📚 Week 1 – Udemy Clone (HTML + Basic CSS)

This project is part of my MERN Stack Development learning journey.
In Week 1, I created a simple Udemy homepage clone using only HTML and basic CSS.
The goal was to understand layout structure, basic styling, and how real websites organize content.


---

🚀 Project Overview

This is a beginner-friendly front-end project where I recreated the core layout of Udemy’s homepage, including:

Header with logo & navigation

Search bar section

Course category blocks / cards

Featured courses layout

Footer section


No frameworks or JavaScript were used — only pure HTML and CSS.


---

🎯 Learning Objectives

Understand the flow and structure of a webpage

Practice HTML tags, hierarchy, and sections

Use CSS for layout, colors, spacing, fonts

Build responsive sections using simple CSS techniques

Improve design thinking by cloning a real website
